# Cursor Configuration Repository

Shared Cursor configurations to standardize AI-assisted development workflows.

## Repository Structure

```
.cursor/
├── rules/          # Project-level rules for code standards and patterns
├── commands/       # Custom commands for common workflows (coming soon)
└── skills/         # Agent skills for specialized capabilities (coming soon)
```

## Quick Links

- [Cursor Rules Documentation](https://cursor.com/docs/context/rules)
- [Cursor Commands Documentation](https://cursor.com/docs/context/commands)
- [Cursor Skills Documentation](https://cursor.com/docs/context/skills)

## Contributing

Add new rules, commands, or skills to the appropriate directory in `.cursor/`. Follow the official documentation for file format and structure requirements.
